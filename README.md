# Clinical Analytics

This is a Django-based project for clinical analytics.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Python
- pip

### Installing

1. Clone the repository
```bash
git clone https://github.com/nsrikanth97/clinical_analytics.git
```

2. Install the dependencies
```bash
pip install -r requirements.txt
```

3. Set up your environment variables. Refer to `clinical_analytics/settings.py` for the required environment variables.

4. Run the server
```bash
python manage.py runserver
```
